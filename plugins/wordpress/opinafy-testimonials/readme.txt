Muestra testimonios verificados de tus clientes con widgets personalizables. Integración sencilla con Opinafy.

== Description ==

**Opinafy** es la plataforma de testimonios en español que te permite recopilar, gestionar y mostrar opiniones verificadas de tus clientes directamente en tu sitio WordPress.

= Características principales =

* **Widget personalizable** - Elige entre múltiples diseños: muro, carrusel, lista, badge y más.
* **Testimonios verificados** - Cada testimonio pasa por un proceso de verificación.
* **Fácil integración** - Instala el plugin, pega tu Project ID y listo.
* **Shortcode flexible** - Usa `[opinafy]` en cualquier página, post o widget.
* **Compatible con Gutenberg** - Funciona con el editor de bloques.
* **SEO friendly** - Los testimonios se renderizan en el DOM para ser indexados.
* **Responsive** - Se adapta perfectamente a móviles y tablets.
* **Rápido** - Script ligero (~25KB) que no afecta la velocidad de tu sitio.

= ¿Cómo funciona? =

1. Crea una cuenta gratuita en [Opinafy](https://opinafy.com/registro).
2. Crea un proyecto y recopila testimonios de tus clientes.
3. Instala este plugin en WordPress.
4. Pega tu Project ID en Ajustes → Opinafy.
5. Los testimonios aparecerán automáticamente en tu sitio.

= Shortcode =

Usa el shortcode `[opinafy]` para mostrar testimonios en cualquier lugar:

* `[opinafy]` - Usa el Project ID configurado en ajustes.
* `[opinafy id="tu-project-id"]` - Especifica un Project ID diferente.

== Installation ==

= Instalación automática =

1. Ve a Plugins → Añadir nuevo en tu panel de WordPress.
2. Busca "Opinafy".
3. Haz clic en "Instalar ahora" y luego en "Activar".
4. Ve a Ajustes → Opinafy y pega tu Project ID.

= Instalación manual =

1. Descarga el archivo ZIP del plugin.
2. Ve a Plugins → Añadir nuevo → Subir plugin.
3. Selecciona el archivo ZIP y haz clic en "Instalar ahora".
4. Activa el plugin.
5. Ve a Ajustes → Opinafy y pega tu Project ID.

= ¿Dónde encuentro mi Project ID? =

1. Inicia sesión en [opinafy.com](https://opinafy.com).
2. Ve a tu panel de proyectos.
3. Selecciona un proyecto.
4. Ve a "Configurar Widget" → "Código de integración".
5. Copia el Project ID que aparece en el código.

== Frequently Asked Questions ==

= ¿Necesito una cuenta de Opinafy? =

Sí, necesitas una cuenta gratuita en [opinafy.com](https://opinafy.com/registro) para crear proyectos y recopilar testimonios.

= ¿El plugin es gratuito? =

Sí, el plugin es 100% gratuito. Opinafy tiene un plan gratuito que incluye hasta 2 proyectos con testimonios ilimitados.

= ¿Afecta la velocidad de mi sitio? =

No. El script del widget es muy ligero (~25KB) y se carga de forma asíncrona (defer), por lo que no bloquea la carga de tu página.

= ¿Funciona con cualquier tema? =

Sí, el widget usa Shadow DOM para aislamiento de estilos, lo que garantiza que se vea bien con cualquier tema de WordPress.

= ¿Puedo personalizar el diseño? =

Sí, desde tu panel de Opinafy puedes elegir entre más de 60 plantillas y personalizar colores, tipografía, diseño y más.

= ¿Los testimonios son indexados por Google? =

Sí, los testimonios se renderizan en el DOM y pueden ser indexados por los motores de búsqueda, lo que beneficia tu SEO.
